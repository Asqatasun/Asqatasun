Pour chaque élément recevant le focus, l’indication visuelle du navigateur ne doit pas être supprimée (propriété CSS outline, outline-color, outline-width, outline-style). Cette règle est-elle respectée ?
Any element with a tabindex can grab the focus
Passed : The "div" element defines a "tabindex" attribute that enables it to grab the focus, no particular definition of the outline css property, the test is passed.
