### Description

> Dans chaque page web, chaque contenu cryptique (art ASCII, émoticon, syntaxe cryptique) vérifie-t-il une de ces conditions ?
> 
> * Un attribut `title` est disponible.
> * Une définition est donnée par le contexte adjacent.
